## v0.5.0 — SPU 五态、类目树、共享包（当前基线，2026-06-03）

### SPU 商品状态（Breaking：枚举扩展）

| 状态值 `SpuStatus` | 中文   | C 端列表 | 可购买       |
| ------------------ | ------ | -------- | ------------ |
| `NOT_LISTED`       | 未上架 | 否       | 否           |
| `ON_SALE`          | 已上架 | 是       | 是（有库存） |
| `SOLD_OUT`         | 已售罄 | 是       | 否           |
| `OFF_SALE`         | 已下架 | 否       | 否           |
| `RESTOCKING`       | 补货中 | 是       | 否           |

- 原 `DRAFT` 废弃，等同 `NOT_LISTED`；seed 会自动迁移历史数据。
- 库存变更后自动同步：已上架 ↔ 已售罄；补货中有库存 → 已上架。
- 实现位置：`packages/shared/src/spu-status.ts`，文档见《共享接口与约定》。

### 管理后台

- 商品列表：**状态列**、状态筛选、行内切换。
- **类目管理**：由扁平表格改为**树形表格**，支持「新增一级类目」「新增子类目」。
- `@simplemall/shared`：开发态直连源码，避免 Vite 预构建缓存缺少新导出（如 `SPU_STATUS_LABEL`）。

### Web 商城

- 商品列表/详情展示售罄、补货中标签；不可购时禁用加购并提示。

### 文档同步

- 架构说明书 v1.1、共享约定、三端开发设计文档、`docs/README.md` 变更规则均已更新。

### 升级注意

```bash
pnpm --filter @simplemall/shared build
cd apps/api && pnpm prisma:seed
./dev.sh stop && ./dev.sh
# Admin 若报 shared 导出错误：
cd apps/admin && rm -rf node_modules/.vite && pnpm dev
```

## v0.8.0 — 标签管理、商城首页与商品详情页（2026-06-03）

### 标签与首页 / 搜索

- **标签表**：`status`（启用/停用）、`isHot`（热门）；Admin 标签管理 CRUD；商品表单仅可选启用标签。
- **API**：`GET /home`、`GET /tags`、`GET /tags/hot`；`GET /spus` 支持 `tag`、`recommend`、`keyword`、统一搜索参数 **`q`**（启用标签名按标签筛，否则按关键词）。
- **Web 顶栏（参考闲鱼）**：仅**首页**展示热门标签；点击标签跳转 **`/search?q=`**，标签无选中回显；搜索框提交同路径。
- **Web 页面**：`pages/search/index.vue` 搜索结果页；Footer；首页热门活动链至搜索页、推荐商品、「猜你喜欢」+ 类目/标签 Tab。
- **样式约定**：`.cursor/rules/web-tailwind.mdc`（Web 强制 Tailwind，非必要不新增 CSS 文件）；`web-responsive.mdc` 与之对齐。

### 商品详情页改造（`pages/products/[id].vue`）

| 模块 | 说明 |
| ---- | ---- |
| **左右布局** | 左侧：图集轮播 + 商品详情富文本（自上而下滚动）；右侧：购买信息区，约占宽 **40%**，`sticky` 吸顶 |
| **图集** | 组件 `ProductImageCarousel`：主图 + `images_json` 轮播，支持箭头/指示点/缩略图 |
| **右侧信息** | 标题/状态/卖点、现价/划线价/优惠金额、配送（运费模板 + 发货地→默认地址）、服务保障标签、规格、数量 |
| **底栏操作** | 固定于右侧栏底部（≤1024px 为全屏底栏）：胶囊按钮（左渐变 + 购物车图标 / 右「立即购买」）+ 收藏 |
| **API 扩展** | `GET /spus/:id` 增加 `express`、`shipFrom`、`serviceGuarantees`（见下） |
| **规格交互** | 商品不可购（非已上架、全 SKU 无库存等）时规格按钮置灰；可购时无库存规格组合单独置灰 |
| **样式实现** | 详情页样式全部由 **Tailwind**（`bp-*` 断点）承担，已移除 `assets/css/product-detail.css` |

### 服务保障

- **数据库**：`service_guarantees` 表；`spus.service_list`（逗号分隔名称，与 `tag_list` 类似）。
- **API**：`GET/POST/PUT/DELETE /admin/service-guarantees`；商品保存校验均为启用项；详情返回 `serviceGuarantees[]`。
- **Admin**：侧栏「服务保障」CRUD；商品表单多选 `ServiceGuaranteeInput`。
- **Seed**：默认「破损包退、假一赔十、退货宝、极速退款」并写入演示商品。

### 其它 Web

- 顶栏/布局样式文件：`header.css`、`layout.css`（首页 `home-content` 最小宽度等）仍保留；详情页不再单独维护 CSS 文件。

迁移包括：`20260603150000_tag_status_hot`（若未执行）、`20260603160000_service_guarantees`（`service_list` 字段）。已部署库仅需 `migrate deploy` + `seed`（seed 会 upsert 服务保障词库）。
