# SimpleMall — Agent 说明

本仓库为轻量电商 Monorepo（API + Nuxt Web + React Admin）。Agent 默认应遵守：

1. **项目规则**：`.cursor/rules/*.mdc`（尤其 `alwaysApply: true`：`version-changelog`、`code-comments`）
2. **文档索引**：`docs/README.md`
3. **版本记录**：有交付价值时更新 `docs/版本迭代说明.md`
4. **接口契约**：`docs/共享接口与约定.md`

本地开发优先 `./dev.sh`；勿在未要求时 `git commit` / `git push`。
